# Expense Tracker 💰

A Python script that tracks and analyses daily expenses.

## What it does
- Shows total expenses and amount spent
- Calculates average and highest expense
- Groups spending by category
- Automatically cleans messy data

## Tech Used
- Python
- Pandas

## Key Feature
Handles messy data automatically — fixes
capital/lowercase issues in categories

## How to Run
```bash
pip install pandas
python expense_tracker.py
```
